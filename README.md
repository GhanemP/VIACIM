# VIA - Customer Journey Intelligence

A customer success operations platform for tracking customer health, engagement, and lifecycle management.

## Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling
- **Inter Font** - Typography

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## Features

- Customer portfolio dashboard with health scores
- Interactive journey timeline visualization
- Engagement gap detection
- AI-powered insights and risk scoring
- Real-time customer health monitoring
